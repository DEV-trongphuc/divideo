=== CANH BAO QUAN TRONG: PHIEN BAN PYTHON KHONG TUONG THICH ===

Ban dang dung Python 3.14. Day la phien ban thu nghiem qua moi, PyTorch chua co ban build ho tro cho Python 3.14 nen ban se bi loi "Could not find a version that satisfies the requirement torch".

👉 CACH KHAC PHUC:
1. Tai va cai dat Python 3.11.9 tu link chinh thuc nay:
   https://www.python.org/ftp/python/3.11.9/python-3.11.9-amd64.exe

2. LUU Y QUAN TRONG KHI CAI DAT:
   Tich chon o vuong "Add Python.exe to PATH" o phia duoi cung cua trinh cai dat truoc khi bam "Install Now".

3. Kiem tra lai:
   Mo mot cua so Terminal moi va go:
   python --version
   (Man hinh phai bao dung chu: Python 3.11.9)

4. Chay lenh cai dat torch ho tro GPU RTX 3050:
   pip install torch torchaudio --index-url https://download.pytorch.org/whl/cu121 --force-reinstall

5. Chay he thong:
   node run_synthesis.js
